# **App Name**: Parallel Lives

## Core Features:

- Global Streaming Discovery: Discover live streams from around the world based on location and category preferences, and provides approximate distances.
- Interactive Global Map: Browse the world through an interactive map, see color-coded pins for live streams, starting soon streams, or recent streams. Find locations where streamers will begin in the future.
- Multi-player Games: Create and join live, interactive games with other users, including trivia, creative challenges, and more.
- Localized Events and Meetups: Find and create local events and meetups, connect with people nearby or globally with shared interests.
- AI-Powered Title Suggestions: Receive AI-generated suggestions for stream titles and categories to improve discoverability, using a tool that helps determine optimal phrasing.
- Community Building: Connect with other users through groups and chats based on shared interests and events.
- Personalized Profile: Customize your profile with a bio, skills, cultural background, and showcase your live streams, highlights, and archived content.

## Style Guidelines:

- Primary background: True Black (#000000) to ensure content takes center stage.
- Secondary background: Dark Grey (#161616) provides depth and separation.
- Surface cards: Card Grey (#212121) for distinct content containers.
- Primary text: Pure White (#FFFFFF) offers high contrast and readability.
- Accent color: Neon Green (#00FFA3) highlights interactive elements and actions.
- Headings: 'Space Grotesk' sans-serif font to communicate a modern, energetic feel. 'Space Grotesk' is suitable for the titles, categories, and featured text across all screens, as well as body copy, should a more minimalistic visual design be desired.
- Minimalist icons with a line art style to maintain a clean and modern aesthetic.
- Four-tab system (Home, Play, Connect, Profile) for clear navigation and content separation.
- Subtle animations, such as button presses (scale 0.95) and tab switches (slide 200ms ease), to enhance the user experience.
